# QBL 🏭

A large-scale multi-page business website developed for **Dreamer Studio** agency. The project covers a full corporate web presence for an industrial/manufacturing company — including product catalogue, technology pages, trade fair references, and more.

## 🔗 Live Demo
[View Live Site](https://oleksandr549.github.io/QBL.github.io/)

## ✨ Features
- Multi-page corporate website (20+ pages)
- Product catalogue with individual product pages
- Technology & machinery sections
- References and case studies
- Trade fair showcase page
- About, Story, and Contact pages
- Cost synergy & planning sections
- Fully responsive layout

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla)
- Custom fonts & assets

## 📁 Project Structure
```
QBL/
├── index.html              # Homepage
├── about.html              # About the company
├── story.html              # Company story
├── contact.html            # Contact page
├── products-list.html      # Full product catalogue
├── product-cart.html       # Product detail / cart
├── technology.html         # Technology overview
├── technology-cliq.html    # Cliq technology page
├── technology-list.html    # Technology list
├── machinery.html          # Machinery section
├── functionality.html      # Functionality overview
├── front-decors.html       # Front decor products
├── shop-display-systems.html # Shop display systems
├── planning.html           # Planning section
├── cost-synergy.html       # Cost synergy page
├── production.html         # Production details
├── references.html         # Client references
├── references-more.html    # Extended references
├── trade-falrs.html        # Trade fairs showcase
├── css/                    # Stylesheets
├── js/                     # JavaScript files
├── img/                    # Images and assets
└── fonts/                  # Custom fonts
```
